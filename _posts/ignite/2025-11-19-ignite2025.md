# Microsoft Ignite 2025: Copilot과 AI 에이전트로 Frontier Firm 시대를 열다

## Microsoft Ignite 2025: Copilot과 AI 에이전트로 Frontier Firm 시대를 열다

Microsoft Ignite는 매년 열리는 Microsoft의 대표 기술 컨퍼런스로, 최신 클라우드, AI, 보안, 생산성 혁신을 발표하는 자리입니다. 올해 Ignite 2025에서는 기업이 AI를 통해 업무 방식을 근본적으로 혁신하고, **Frontier Firm**—인간 중심, 에이전트 운영 조직—으로 도약할 수 있도록 돕는 새로운 기능과 전략을 공개했습니다.

![ignite](/mwkorea/assets/images/20251119/image01.png)

---

## 주요 기능 업데이트 5줄 요약

1. **Work IQ 출시**: Copilot이 사용자·조직·업무 패턴을 이해하는 지능 레이어, 커스텀 에이전트 개발 가능 **(GA)**. 
2. **Office 앱 Agent Mode 확장**: Word·Excel 일반 제공(GA), PowerPoint는 Frontier Program에서 제공 중. 
3. **음성 기반 Copilot**: 모바일 앱에서 음성 대화 가능, Outlook 음성 기능은 퍼블릭 프리뷰 진행 중 (12월 GA 예정). 
4. **Sora 2 영상 생성 AI와 Teams 협업 에이전트**: Facilitator Agent GA, 서드파티 연동은 프리뷰 중. 
5. **Agent 365 관리 플랫폼과 Windows AI 기능**: 에이전트 등록·보안·모니터링 제공, Windows Taskbar Copilot 및 Windows 365 for Agents는 퍼블릭 프리뷰 진행 중.

## 업무의 새로운 지능, Work IQ로 완성된 Copilot

Microsoft 365 Copilot은 이제 업무에서 없어서는 안 될 핵심 도구로 자리 잡았습니다. 이미 포춘 500대 기업의 90% 이상이 Copilot을 활용하고 있으며, 지난 1년 동안 400개 이상의 새로운 기능을 선보이며 혁신을 가속화했습니다. 최신 GPT-5 모델부터 컴퓨터 사용(CUA), Sora 2까지, 업무에 최적화된 AI 모델을 지속적으로 도입하고 있으며, Anthropic 모델을 추가해 다양한 업무 요구에 맞는 최적의 모델을 제공합니다.

Work IQ는 Copilot과 에이전트의 지능을 뒷받침하는 핵심 기능으로, 사용자의 역할과 조직을 깊이 이해합니다. 이메일, 파일, 회의, 채팅 등 업무 전반의 데이터를 기반으로 업무 맥락을 파악하고, 개인의 스타일과 선호도, 습관을 학습해 조직도를 넘어 ‘업무 관계도’를 이해합니다. 데이터와 메모리를 결합한 추론 기능은 통찰을 제공하고 다음 최적의 행동을 예측하며, Copilot은 사용자의 의도에 맞는 적절한 에이전트를 추천합니다. Work IQ는 Word, Outlook, Teams 등 일상적으로 사용하는 앱에 깊이 통합되어 있어 Copilot이 지속적으로 학습하고, 이를 기반으로 더욱 정교하고 개인화된 경험을 제공합니다.

또한 Work IQ는 맞춤형 에이전트 개발에도 활용됩니다. Copilot Studio나 API를 통해 보안이 강화된 에이전트를 구축할 수 있으며, 기존 권한, 민감도 레이블, 규정 준수, 감사 및 정책을 모두 준수합니다. Microsoft 365 Copilot 라이선스 또는 사용량 기반 과금으로 이용 가능하며, Copilot과 에이전트는 조직의 모든 지식을 업무 성과로 연결합니다. Work IQ 기반 커스텀 에이전트는 **Copilot Studio 및 API에서 일반 제공(GA)** 중입니다.

## Agent Mode부터 음성 인터페이스까지, 새로운 차원의 생산성

Office 앱에서 Agent Mode를 통해 Copilot Chat 또는 앱 내에서 세계적 수준의 문서, 프레젠테이션, 스프레드시트를 생성할 수 있습니다. Excel에서는 Anthropic과 OpenAI 모델 중 선택 가능하며, Word는 일반 제공(GA), PowerPoint는 Frontier 프로그램을 통해 제공됩니다. Word, Excel, PowerPoint용 새로운 에이전트는 Copilot Chat에서 고품질의 문서와 콘텐츠를 제작할 수 있도록 지원합니다.

Copilot은 이제 음성 인터페이스를 통해 동료처럼 대화할 수 있습니다. 휴대폰에서 “오늘의 주요 업무는?” 또는 “놓친 회의 요약해줘”라고 말하면 Copilot이 바로 응답하며, Outlook에서는 음성으로 메일과 일정 관리가 가능합니다. ‘요약 후 회신’과 같은 원터치 프롬프트도 지원해 이메일 처리와 일정 관리가 더욱 간편해집니다.

Copilot Chat은 모든 Microsoft 365 구독자에게 추가 비용 없이 제공되는 안전한 업무용 AI 채팅으로, 곧 Agent Mode와 Outlook 통합 기능이 추가될 예정입니다. 또한 Sora 2는 차세대 AI 영상 제작 기능을 제공하여 마케팅 및 소셜 콘텐츠용 숏폼 비디오를 손쉽게 제작할 수 있습니다. Edge for Business는 Copilot Mode 활성화 시 세계 최초의 안전한 엔터프라이즈 AI 브라우저로 진화하며, Teams Mode는 Copilot의 1:1 채팅을 Teams 그룹 채팅으로 확장해 협업을 강화합니다. Facilitator Agent는 회의 진행, 노트 작성, 액션 관리까지 지원하며, MCP 서버를 통해 GitHub, Asana, Jira 등 서드파티 앱과도 연동됩니다. **Agent Mode는 Word와 Excel에서 GA, PowerPoint는 Frontier 프로그램에서 제공 중이며, 음성 인터페이스는 12월 GA, Outlook 음성 기능은 퍼블릭 프리뷰 상태입니다. Copilot Chat의 확장 기능은 2026년 초 프리뷰로 제공됩니다.**

## CRM부터 워크플로우까지, Copilot 주도하는 변화

Copilot은 비즈니스 프로세스를 근본적으로 변화시키는 창구입니다. Sales Development Agent는 영업팀의 역량을 확장해 파이프라인 구축, 리드 육성, 맞춤형 아웃리치를 자동화하며, Salesforce 및 Dynamics 365와 연동되어 Microsoft 365 앱 내에서 바로 실행됩니다. App Builder와 Workflows Agent는 자연어를 통해 앱과 워크플로우를 손쉽게 생성하고 배포할 수 있도록 지원합니다.

Copilot Studio는 OpenAI와 Anthropic 모델을 활용해 업무에 최적화된 에이전트를 개발할 수 있으며, 최신 GPT-5.1 모델도 추가되었습니다. Microsoft Agent Factory는 Foundry와 Copilot Studio를 기반으로 에이전트를 빠르고 안전하게 구축·배포할 수 있는 단일 요금제 모델을 제공하며, Workforce Insights Agent는 리더에게 실시간 조직 데이터를 제공해 데이터 기반 의사결정을 지원합니다. **Sales Development Agent는 2025년 12월 Frontier 프로그램에서 제공되며, App Builder와 Workflows Agent는 Frontier 프로그램에서 제공 중입니다. Agent Factory는 GA 상태입니다.**

## 모든 에이전트를 한눈에, Agent 365

IDC는 2028년까지 13억 개 이상의 에이전트가 존재할 것으로 전망합니다. Agent 365는 이러한 에이전트를 안전하게 배포·관리·거버넌스할 수 있는 통합 플랫폼으로, 조직 내 모든 에이전트를 단일 소스로 관리하고 작업별 최소 권한을 부여합니다. 또한 에이전트, 사용자, 데이터 간 연결을 시각화하고 실시간 모니터링을 제공하며, 앱과 데이터 연동을 통해 인간-에이전트 워크플로우를 간소화합니다. 보안 기능은 위협 탐지 및 공격 대응을 강화해 조직의 안전을 보장합니다. **Agent 365는 Frontier 프로그램을 통해 제공됩니다.**

## AI와 Windows의 만남, 새로운 가능성

Windows는 AI 시대를 위한 새로운 기반을 제공합니다. Copilot on Taskbar는 음성 또는 텍스트로 앱, 파일, 설정을 검색하고 에이전트를 실행할 수 있는 진입점을 제공하며, Windows 365 for Agents는 클라우드 기반 보안 환경에서 엔터프라이즈급 에이전트 개발 및 배포를 지원합니다. Manus 등 선도 기업들은 이미 이 기능을 활용해 차세대 AI 솔루션을 제공하고 있습니다. **Copilot on Taskbar는 퍼블릭 프리뷰 상태이며, Windows 365 for Agents는 퍼블릭 프리뷰로 제공됩니다.**

---

## 보안부터 SMB까지, AI 혁신의 확장

Microsoft는 보안 전문가를 위해 Microsoft Defender, Entra, Intune, Purview에 12개의 신규 보안 에이전트를 추가했습니다. Security Copilot은 모든 Microsoft 365 E5 고객에게 제공되며, SMB를 위한 Copilot Business는 300명 이하 기업을 대상으로 12월부터 사용자당 월 $21에 제공됩니다. **Security Copilot은 오늘부터 단계적으로 배포되며, Copilot Business는 2025년 12월 GA 예정입니다.**


Microsoft는 Copilot과 에이전트를 통해 모든 기업이 Frontier Firm으로 도약할 수 있도록 지원합니다. 더 자세한 정보는 https://www.microsoft365.com/copilot에서 확인하세요.
