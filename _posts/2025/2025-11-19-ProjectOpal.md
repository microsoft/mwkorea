# 복잡한 작업을 단순하게, Project Opal의 힘

## Microsoft 365 Copilot의 새로운 기능, Project Opal

기업은 매일 반복적이고 수동적인 작업을 처리합니다. 이런 업무는 필수적이지만, 전략적 사고와 창의적인 활동을 방해하고 시간을 빼앗습니다. 예를 들어, 컴플라이언스 감사 자료를 여러 사이트에서 수집하고 정리하는 일, 신규 직원의 장비 주문과 시스템 접근 권한 설정, 구매 주문 검증 등은 모두 중요한 작업이지만 효율적이지 않습니다.

이제 **Microsoft 365 Copilot에 새롭게 추가된 Project Opal**이 그 문제를 해결합니다. AI 기반으로 설계된 이 기능은 안전하고 투명한 환경에서 작업을 자동으로 실행하며, 사용자가 완전히 제어할 수 있습니다.

## Project Opal이 제공하는 가치

Frontier 참여 고객들은 이미 다음과 같은 효과를 경험하고 있습니다:

- **컴플라이언스 감사 자동화**: 필요한 웹사이트 탐색, 스크린샷 캡처 및 정리, 문서 검증까지 자동 처리. Microsoft 엔지니어들은 감사 작업에서 주당 최대 20시간을 절약했습니다. 
- **승인 프로세스 간소화**: 그룹 멤버십 요청과 같은 다단계 승인 워크플로우를 자동화. 
- **HR 업무 단순화**: 근태 기록 제출, 휴가 신청 등 반복적인 HR 작업을 손쉽게 처리. 
- **온보딩 프로세스 최적화**: 신규 직원을 배포 리스트에 추가하고, 장비를 주문하며, 시설팀에 알림을 보내는 작업을 자동화해 일관된 시작을 보장.

---

## 작동 방식

Project Opal은 Microsoft 365 Copilot 앱에서 **Frontier 모듈**을 통해 접근할 수 있습니다. 새로운 작업을 시작하려면 ‘New’를 선택하고 원하는 업무를 설명한 뒤 **Start**를 클릭하세요. 그러면 Opal이 계획을 수립하고, 안전한 Windows 365 Cloud PC를 실행해 고급 추론 모델과 컴퓨터 사용(Computer Use) 기능을 통해 작업을 진행합니다.

실시간으로 작업을 모니터링할 수 있으며, 언제든지 중지하거나 직접 제어할 수 있습니다. 새로운 명령을 입력하면 Opal은 즉시 계획을 업데이트합니다.

## Project Opal의 핵심 기능

업무 자동화와 고급 추론을 결합해, 사용자가 완전한 제어권을 가진 안전한 작업 환경을 제공합니다

1. 작업 중심 경험
업무의 본질에 집중할 수 있도록 설계되었습니다. 새로운 작업을 시작하거나 이전 작업을 재실행하고, 추천 작업을 선택하는 모든 과정이 직관적입니다. 모든 작업은 실시간으로 관찰 가능하며, 사용자가 언제든지 개입할 수 있어 완전한 제어권을 제공합니다.
2. 고급 추론 모델
단순한 명령을 실행하는 수준을 넘어, 요청을 분석해 동적인 계획을 수립합니다. 필요한 도구를 적절히 조합하고, 상황 변화에 따라 유연하게 대응해 업무를 끝까지 완수합니다.
3. 컴퓨터 사용 자동화
Microsoft Edge를 활용해 클릭, 입력, 탐색을 사람처럼 수행합니다. 모든 작업은 Windows 365 Cloud PC에서 안전하게 실행되며, 조직에서 허용한 사이트만 접근합니다. 사용자는 언제든지 제어권을 되찾을 수 있습니다.
4. 실시간 관찰 가능성
작업 진행 상황을 실시간으로 확인하거나, 나중에 상세 로그와 재생 기능을 통해 검토할 수 있습니다. 계획, 실행 화면, 활동 타임라인까지 투명하게 제공되어 신뢰성을 높입니다.
5. 사용자 개입 가능성
민감한 정보 입력이 필요할 때는 작업을 자동으로 중지하고 사용자에게 알림을 보냅니다. 사용자가 개입 후 다시 이어서 작업을 진행할 수 있어 안전성과 효율성을 모두 확보합니다.
6. 엔터프라이즈급 보안
모든 작업은 Entra 가입, Intune 등록된 전용 Cloud PC에서 실행됩니다. 관리자 제어 기능과 허용 목록 기반 접근 정책을 통해 조직의 보안 요구사항을 충족합니다.

---

## Project Opal의 보안 및 관리 기능

기업 환경에서 요구되는 최고 수준의 보안과 투명성을 제공하며, 모든 자동화 작업을 안전하게 관리합니다.

1. 전용 Cloud PC 환경
모든 작업은 Entra 가입, Intune 등록된 Windows 365 Cloud PC에서 실행됩니다. 이를 통해 기업의 보안 정책을 준수하며, 작업이 안전하게 격리된 환경에서 진행됩니다.
2. 실시간 감독 기능
AI 모델의 행동을 실시간으로 모니터링하고, 가드레일을 적용하며, 필요한 경우 사용자 입력을 요청합니다. 이는 자동화 과정에서 발생할 수 있는 위험을 최소화하고 신뢰성을 높입니다.
3. 완전한 사용자 제어
사용자는 언제든지 작업을 중지하거나 직접 제어할 수 있습니다. 민감한 정보 입력이 필요한 경우, Opal은 자동으로 작업을 일시 중지하고 사용자에게 알림을 제공합니다.
4. 허용 목록 기반 접근
기본적으로 Opal은 어떤 사이트에도 접근할 수 없습니다. 관리자가 지정한 도메인만 허용되며, 이는 Microsoft Edge 정책을 통해 적용됩니다. 기업은 필요한 범위 내에서만 안전하게 작업을 수행할 수 있습니다.
5. 투명한 실행과 기록
모든 작업과 메시지는 사용자에게 투명하게 공개됩니다. 작업 로그와 활동 타임라인을 통해 언제든지 실행 내역을 검토할 수 있어, 보안과 감사 요구사항을 충족합니다.

## 시작하기
Project Opal은 Microsoft 365 Copilot 고객을 대상으로 Frontier 프로그램을 통해 제공됩니다. 현재는 초기 단계이며, 앞으로 지속적으로 발전할 예정입니다. 새로운 업무 자동화 경험을 함께 만들어갈 준비가 되어 있다면, 지금이 바로 기회입니다.
