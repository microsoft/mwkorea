# Microsoft 365 Copilot 2026년 6월 업데이트: Cowork 정식 출시와 한 달간의 변화 총정리

Microsoft는 매달 Microsoft 365 Copilot의 새 기능과 개선 사항을 정리해 공유합니다. 2026년 6월 에디션의 헤드라인은 단연 Copilot Cowork의 전 세계 정식 출시(GA)입니다. 하지만 그 외에도 Word·Excel·PowerPoint·Outlook 전반의 편집·개인화 기능, Copilot Agents의 확장, 그리고 IT 관리자를 위한 비용 관리·보안 강화까지 이번 한 달에 담긴 변화가 상당히 많습니다.

이번 글에서는 6월 한 달간 발표된 사용자 기능과 IT 관리자 기능을 한 번에 정리합니다.

---

## 사용자 기능 (User capabilities)

### Copilot Cowork 전 세계 정식 출시

계획하고, 실행하고, 실제 업무를 완결하는 에이전트 시스템 Copilot Cowork가 전 세계 정식 출시되었습니다. 사용자가 작업을 정의하면 Cowork가 처음부터 끝까지 실행해, 초안이나 제안이 아니라 완성된 결과물을 돌려줍니다. Work IQ 기반으로, Cowork는 모든 작업을 조직이 이미 운영하는 시스템에 근거하게 만들어 실제 맥락을 반영합니다.

Cowork는 사용량 기반 과금(usage-based billing)으로 활성화되며, 관리자는 Microsoft 365 관리 센터의 새로운 비용 관리 대시보드(Cost Management Dashboard)로 크레딧 사용을 모니터링하고 예산을 관리하며 지출을 통제할 수 있습니다.

Cowork 플러그인 생태계에 대한 자세한 내용은 별도 글 [워크플로 자체가 제품이다: Copilot Cowork 플러그인이 바꾸는 비즈니스 AI](/mwkorea/copilot/CoworkPlugins/)에서 다뤘습니다. 이번 달 Cowork에 함께 추가된 세부 기능은 다음과 같습니다.

- **Chat/Cowork 전환 토글**: Microsoft 365 Copilot 앱에서 답변 작업(Chat)과 실행 작업(Cowork) 사이를 손쉽게 전환
- **작업에 맞는 모델 선택**: PowerPoint·그래픽 같은 시각 작업에는 Anthropic 모델을, 인용이 필요한 심층 리서치에는 GPT 5.5를 Cowork가 상황에 맞게 선택 (Frontier 제공)
- **플러그인 확대**: Enosix, Harvey, LSEG, Miro, monday.com, Moody's, Morningstar, S&P Global Energy, TeamsMaestro가 플러그인 스토어에 새로 추가되었고, Databricks는 사이드로딩으로 제공. Microsoft Fabric과 Dynamics 365 전체 포트폴리오(Sales, Customer Service, ERP 앱 등)도 지원
- **커스텀 스킬 생성·관리**: Customize 탭에서 반복 작업을 위한 커스텀 스킬을 직접 만들고 재사용하며, 파트너 제공 스킬·조직 전체 스킬과 결합 가능
- **비주얼 생성·편집**: OpenAI의 ChatGPT Images 2.0 기반으로 덱 그래픽·문서 삽화·이메일 이미지를 직접 만들고 편집. 조직의 브랜드 PowerPoint 템플릿도 활용해 승인된 색상·폰트·레이아웃으로 덱 생성
- **Edge 브라우저 활용**: Cowork가 Edge를 사용해 조직의 비즈니스 시스템·웹사이트·인트라넷 사이트 전반에서 작업 수행 (Frontier 제공)
- **모바일 지원**: iOS·Android에서 Cowork 사용 가능, 기기 간 동일한 맥락과 대화 유지
- **푸시 알림**: 승인, 입력 요청, 완료 등 장시간 작업에 대한 알림 제공

### Copilot 앱·Chat의 인용·추론 강화

- **Power BI 데이터 추론**: Work IQ를 통해 Copilot이 Power BI 엔터프라이즈 데이터를 추론해, 리포트·시맨틱 모델에서 근거 있는 답변을 직접 제공 (Frontier)
- **Dataverse 검색·질의**: Copilot Chat에서 자연어로 Dataverse 비즈니스 데이터를 조회 가능 (6월 공개 미리 보기, 9월 정식 출시 예정)
- **딥 시테이션(Deep citations)**: 답변을 파일 전체가 아니라 참조된 파일의 특정 부분에 직접 연결. Word·PowerPoint 지원으로 시작, 이후 회의·웹·PDF 참조로 확장 예정
- **재생성(Regenerate) 액션**: "다시 시도" 또는 "모델 전환"으로 최신 프롬프트에 대한 대안 응답을 빠르게 받는 기능
- **Microsoft 365 Copilot의 Vision**: 화면이나 휴대폰 카메라로 보이는 것을 Copilot에 보여주고 음성으로 대화하며 질문 (7월 순차 제공)
- **Copilot Page 제안 편집**: Copilot Shortcuts 메뉴의 "Suggested edits"로 페이지 콘텐츠에 대한 피드백과 개선 제안을 받아 바로 적용

### Copilot Notebooks 확장

- Outlook 이메일을 Copilot Notebooks의 참조 자료로 추가 가능 (7월 제공 예정)
- 기존에는 Microsoft 365 Copilot 라이선스 사용자 전용이던 Copilot Notebooks가 Copilot Chat 사용자에게도 제공되어, 마인드맵·학습 가이드 등으로 활용 가능

### Outlook의 Copilot 커스터마이징

- Classic Outlook for Windows에서 Copilot 설정에 직접 접근 가능 (7월 제공 예정)
- 작성 화면에서 초안 일부를 선택해 길이·톤·구조를 바로 조정 (웹에서 6월 제공)

### Word의 Copilot 문서 편집 개선

- **모델 선택**: Word의 Copilot 편집에서 Anthropic 모델 선택 가능, 재작성·요약·재구성 작업의 유연성 향상
- **Copilot Catchup**: 마지막으로 열어본 이후 문서에서 무엇이 바뀌었는지 콘텐츠 카드로 빠르게 요약
- **이미지 생성·삽입**: Copilot으로 이미지를 만들어 편집 흐름 안에 바로 삽입
- **iOS 지원**: iPhone·iPad에서도 Word의 Copilot 에이전틱 편집 기능 사용 가능
- **대화 기록 유지**: Copilot Chat 대화 내역이 Word 앱까지 이어져 연속적인 문서 작업 지원
- **댓글 기반 편집**: 댓글에 담긴 피드백을 Copilot이 추론해 문서에 직접 반영

### PowerPoint의 브랜딩·스킬·참조 업데이트

- **Brand Kit Picker**: 관리자가 승인한 브랜드 키트를 선택해 색상·폰트·비주얼 스타일이 처음부터 맞춰진 프레젠테이션 제작
- **재사용 가능한 스킬**: 반복되는 프레젠테이션 작업을 스킬로 저장해 재사용 (웹에서 6월 제공)
- **SharePoint·OneDrive 참조**: 프레젠테이션 제작 시 SharePoint 라이브러리·OneDrive 폴더를 참조해 조직의 기존 콘텐츠에 근거한 덱 제작

### Excel의 스킬·개인화·규칙

- **Skills**: 반복 가능한 워크플로를 재사용 인스트럭션으로 패키징
- **Personalization**: 서식·명명 규칙·수식·피벗테이블·리포트 스타일에 대한 선호를 한 번 설정하면 모든 워크북에 일관되게 적용
- **워크북 규칙 시트(.Rules sheet)**: 구조·서식·명명·수식 규칙과 예시를 특정 워크북에 저장해, 해당 파일을 편집하는 모든 사람이 동일한 가이드를 따르도록 함

### Copilot Agents: Planner Agent와 Researcher 옵션

- Researcher 에이전트를 Copilot 채팅에 추가하면, 지원되는 Researcher 모델·모드를 대화 안에서 직접 선택 가능
- **Planner Agent**: Microsoft 365 Copilot 안에서 인터랙티브 작업 카드와 실행 가능한 인사이트를 제공해 Planner 계획의 작업 우선순위 지정을 지원. 자연어로 작업을 만들고 갱신하거나, 목표·버킷을 갖춘 구조화된 계획 생성 요청 가능

## IT 관리자 기능 (IT admin capabilities)

### Microsoft 365 관리 센터의 신규 도입·관리 기능

- **조직 프롬프트(Organization prompts)**: 관리자가 관리 센터에서 조직 프롬프트를 만들고 게시해, 사용자가 Copilot Chat 프롬프트 제안과 Prompt Gallery에서 발견하도록 안내
- **비용 관리 대시보드(Cost Management Dashboard)**: Cowork를 포함한 사용량 기반 과금 AI 경험의 활성화·관리·지출 최적화를 위한 중앙 대시보드. 크레딧 할당, 그룹별 지출 정책·한도 설정, 선불/종량제 선택, 리포팅·예산·알림·상한선 지원
- **Insights 팀별 비용 리포팅**: 직속 부하 5명 이상 관리자, Insights 분석가, 글로벌 관리자를 대상으로 팀 단위 Copilot 크레딧 사용 현황 대시보드 제공 (기본 활성화, 7월 제공 예정)
- **에이전트 지표(Agent metrics)**: Insights에서 세부 에이전트 사용 데이터와 조직 맥락을 결합해 도입 추이 파악·맞춤 리포트 작성 가능 (7월 공개 미리 보기, 9월 정식 출시)
- **하이브리드 조인 디바이스 지원**: 온프레미스 Active Directory와 Microsoft Entra ID에 모두 연결된 디바이스에도 조직 메시지(Organizational Messages) 전달 가능
- **Vision 제어**: 화면·카메라 공유 스트림을 모델 입력으로 쓰는 Microsoft 365 Copilot의 Vision 기능에 대한 관리자 통제 제공

### Purview Data Loss Prevention의 이메일 제한·Cowork 확장

- **외부 이메일 참조 제한**: Microsoft Purview DLP가 외부 발신자의 메시지를 Copilot 응답의 요약·참조·근거 데이터로 사용하지 못하도록 제한. Microsoft 365 Copilot과 Copilot Studio에서 만들어 Microsoft 365 Copilot에 게시된 에이전트에 적용 (6월 공개 미리 보기, 7월 정식 출시 예정)
- **Cowork로 확장된 Purview 컨트롤**: 민감도 레이블 상속·표시, 감사 로깅, DSPM 작업 탐색기의 상호작용 콘텐츠, Insider Risk Management, eDiscovery, Data Lifecycle Management(DLM), Communication Compliance가 모두 Cowork 상호작용에도 적용

## 이번 달 업데이트가 말해주는 것

6월의 흐름을 관통하는 키워드는 명확합니다. "Copilot이 대화형 도우미에서 실행형 동료로" 옮겨가고 있다는 것입니다. Cowork의 정식 출시는 그 상징적인 장면이고, Word·Excel·PowerPoint의 스킬·개인화 기능은 반복 업무를 표준화하는 방향이며, 관리자용 비용 관리·Purview 확장은 그 실행형 AI를 기업이 통제 가능한 방식으로 운영하기 위한 안전장치입니다.

## 한국 조직을 위한 시사점

- **Cowork 비용 거버넌스부터 준비**: Cowork는 사용량 기반 과금이므로, 도입 전에 비용 관리 대시보드로 예산·정책을 먼저 설계하는 것이 안전합니다.
- **팀별 사용 현황 가시성 확보**: 직속 부하 5명 이상 관리자에게 제공되는 팀별 크레딧 리포팅을 활용해, 부서별 AI 활용 수준을 점검하세요.
- **문서 도구별 브랜드·규칙 표준화**: PowerPoint의 Brand Kit Picker, Excel의 워크북 규칙 시트, Word의 스킬 재사용 기능을 조직 표준 템플릿과 연결하면 일관성이 크게 개선됩니다.
- **외부 메일 데이터 유출 방지**: Purview DLP의 외부 이메일 참조 제한 기능은 정식 출시(7월) 전에 정책 적용 범위를 미리 검토해 두면 좋습니다.
