# Microsoft 365 Copilot에 Anthropic Claude Fable 5 추가 — 모델 선택의 폭이 넓어진다

Microsoft가 Microsoft 365 Copilot의 모델 선택지를 또 한 번 넓혔습니다. 이번에 추가된 모델은 Anthropic의 최신작인 **Claude Fable 5**로, 프리뷰(preview) 형태로 제공됩니다. 단순한 한 번의 프롬프트를 넘어, **길고 여러 단계로 이어지는(multi-step) 업무 흐름**을 지원하도록 설계된 것이 특징입니다.

조직의 업무가 "한 줄 질문에 한 줄 답변"에서 점점 더 복잡한 작업으로 옮겨가면서, 많은 고객이 Copilot을 활용해 **계획을 세우고(plan), 초안을 작성하고(draft), 여러 차례 다듬는(iterate)** 작업을 하고 있습니다. 여기에 **Work IQ**가 결합되면 Copilot이 파일·회의·채팅·비즈니스 데이터 같은 실제 업무 맥락에 답변을 근거(ground)하게 되어, 결과물이 지금 하고 있는 일에 한층 더 들어맞게 됩니다.

이번 통합은 "최신 AI 혁신을 빠르면서도 책임감 있게(responsibly) Microsoft 365 Copilot에 가져온다"는 Microsoft의 방향을 다시 한 번 보여줍니다. 엔터프라이즈 보호 장치, 관리자 제어, 그리고 고객 선택권이 모두 기본으로 내장되어 있습니다.

---

## 무엇이 새로운가

- **Claude Fable 5 (preview) 추가**: Anthropic의 최신 모델이 Microsoft 365 Copilot의 모델 선택지에 새로 들어왔습니다.
- **복잡한 업무 흐름 지향**: 짧은 단발성 질의가 아니라, 여러 단계에 걸친 길고 복잡한 작업을 더 잘 처리하도록 설계되었습니다.
- **Work IQ와의 결합**: 파일·회의·채팅·업무 데이터 등 실제 업무 맥락에 근거한 응답으로, 결과물의 관련성이 높아집니다.
- **모델 선택권 강화**: 작업 성격에 따라 "딱 맞는 모델(right model for the job)"을 골라 쓸 수 있는 폭이 넓어졌습니다.

## 어디에서 쓸 수 있나

현재 Claude Fable 5(preview)는 다음과 같은 형태로 제공됩니다.

- **Copilot Cowork (Frontier)**: 기본적으로 꺼져 있는(default-off) 옵션으로 제공됩니다. 즉, 관리자가 명시적으로 켜야 사용할 수 있습니다.
- **Copilot in Excel / PowerPoint**: 프라이빗 프리뷰(private preview) 형태로 오늘부터 사용 가능합니다.

작업에 가장 알맞은 모델을 직접 선택해 사용할 수 있다는 점이 핵심입니다.

## 관리자·도입 관점 체크포인트

- **기본 비활성(default-off)**: Copilot Cowork에서 Claude Fable 5는 기본적으로 꺼져 있습니다. 관리자는 조직의 정책과 준비 상태에 맞춰 접근 권한을 직접 관리할 수 있습니다.
- **데이터 보존(data retention) 고려**: Claude Fable 5의 사용은 **Anthropic의 데이터 보존 정책**의 적용을 받습니다. 따라서 조직은 모델을 활성화하기 전에 이 부분을 도입 의사결정의 한 요소로 반드시 검토해야 합니다.
- **지역·테넌트별 가용성 차이**: 가용성은 지역과 테넌트 구성에 따라 달라질 수 있습니다. 최신 롤아웃 상태는 Microsoft 365 Roadmap과 Microsoft 365 Copilot 릴리스 노트에서 확인하세요.

## 시작하기

Microsoft 365 Copilot 안에서 작업에 맞는 모델을 고르면 됩니다. Claude Fable 5(preview)는 Copilot Cowork(Frontier)에서는 기본 꺼짐 옵션으로, Excel·PowerPoint에서는 프라이빗 프리뷰로 제공됩니다.

- 모델 활성화 및 정책 관리는 관리자가 조직의 준비 상태에 맞춰 진행합니다.
- 모델 자체에 대한 더 자세한 내용은 [Anthropic의 Claude Fable 5 발표](https://www.anthropic.com/news/claude-fable-5-mythos-5)를 참고하세요.
- Microsoft의 Anthropic 모델 통합에 대한 안내는 [aka.ms/AnthropicModelsMSFT](https://aka.ms/AnthropicModelsMSFT)에서 확인할 수 있습니다.

## 마무리

이번 Claude Fable 5 추가는 Microsoft 365 Copilot이 **단일 모델에 묶이지 않고, 작업 성격에 따라 최적의 모델을 선택하는 방향**으로 계속 진화하고 있음을 보여줍니다. 특히 길고 복잡한 다단계 작업이 많은 조직이라면, Work IQ 기반의 업무 맥락 근거와 결합된 이 모델이 의미 있는 선택지가 될 수 있습니다. 다만 Anthropic의 데이터 보존 정책이 적용된다는 점은 도입 전에 반드시 짚어봐야 할 부분입니다.
