# Edge의 Rewrite by Copilot, 기업 데이터 보호 기준을 더한다

웹에서 글을 쓰는 순간은 생각보다 많습니다. 고객에게 보낼 짧은 메시지, 내부 포털에 남기는 공지, SaaS 도구 안의 설명 문구, 양식의 자유 입력란까지 말이죠. 이런 상황에서 Microsoft Edge의 **Rewrite by Copilot**은 열린 웹 페이지 안에서 사용자가 입력한 텍스트를 톤, 형식, 길이에 맞게 다듬도록 돕는 기능입니다.

Microsoft 365 Roadmap 항목 RM420335에 따르면, 이 Rewrite by Copilot 기능이 **상업용 고객을 위한 기업 데이터 보호(Enterprise Data Protection) 기준**을 포함하도록 업데이트됩니다. 사용자가 **Microsoft Entra ID**로 로그인한 상태에서 Microsoft Edge의 Rewrite 기능을 사용할 때, 기업 데이터 보호가 적용되는 방향입니다.

다만 중요한 변경 사항도 함께 공지되었습니다. Microsoft는 2026년 3월 20일 업데이트에서 이 기능의 출시를 **일시 중단(paused)**했으며, 새 출시 일정은 추후 공지하겠다고 밝혔습니다. 로드맵상 GA는 2026년 8월로 표시되어 있지만, 실제 일정은 변경될 수 있습니다.

---

## 무엇이 새로운가

- **Rewrite by Copilot의 보호 수준 강화**: Edge 안에서 입력 중인 텍스트를 Copilot으로 다시 쓰는 기능에 상업용 고객을 위한 기업 데이터 보호 기준이 포함됩니다.
- **Microsoft Entra ID 기반 보호**: 사용자가 Microsoft Entra ID로 로그인한 경우, Rewrite 기능 사용 시 기업 데이터 보호가 적용됩니다.
- **웹 페이지 안에서 바로 편집**: 별도 앱으로 복사해 이동하지 않고, Edge에서 열려 있는 페이지 안의 입력 텍스트를 톤·형식·길이에 맞게 다듬을 수 있습니다.
- **출시 일시 중단**: Microsoft는 이 기능의 릴리스를 임시로 중단했으며, 새 일정은 추후 발표 예정입니다.

---

## 왜 중요한가

Copilot 기반 문장 재작성은 생산성을 크게 높일 수 있지만, 기업 환경에서는 항상 같은 질문이 따라옵니다.

> 이 텍스트가 어디로 가는가? 회사 데이터가 안전하게 보호되는가?

이번 업데이트의 핵심은 바로 이 지점입니다. Edge 브라우저 안에서 글을 다듬는 경험을 제공하면서도, 상업용 고객에게 필요한 기업 데이터 보호 기준을 함께 적용하려는 방향입니다. 특히 사용자가 Entra ID로 로그인한 상태를 기준으로 보호가 적용된다는 점은, 조직 계정 기반의 보안·거버넌스 체계와 자연스럽게 연결됩니다.

---

## 활용 시나리오

- **고객 응대 문구 다듬기**: 웹 기반 CRM이나 지원 포털에서 작성 중인 답변을 더 정중하거나 간결하게 변경
- **내부 포털 글쓰기**: 인트라넷 공지, 지식 문서, 정책 안내 문구를 더 명확한 톤으로 재작성
- **SaaS 업무 도구 입력 지원**: 프로젝트 관리 도구, 설문, 승인 요청, 티켓 시스템 안에서 작성 중인 텍스트를 상황에 맞게 조정
- **비영어권 사용자 지원**: 영어 문장을 더 자연스럽게 다듬거나, 긴 문장을 짧게 정리하는 데 활용

## 도입 관점 체크포인트

- **사용자 안내는 일정 확정 후**: 기능 출시가 일시 중단된 상태이므로, 대규모 사용자 공지는 새 일정이 공지된 뒤 진행하는 것이 안전합니다.
- **Edge Copilot 정책 검토**: 조직에서 Edge의 Copilot 관련 기능을 어떻게 허용·제한하고 있는지 관리 정책을 점검하세요.
- **데이터 보호 메시지 정리**: 사용자에게 "Entra ID 로그인 상태에서 기업 데이터 보호가 적용된다"는 조건을 명확히 안내할 필요가 있습니다.
- **웹 기반 업무 도구와 함께 검토**: 이 기능은 Edge 안에서 열린 페이지의 입력 텍스트를 다루므로, 조직에서 많이 쓰는 웹 업무 도구와 함께 사용자 경험을 확인하면 좋습니다.

---

## 마무리

Rewrite by Copilot의 기업 데이터 보호 적용은, 브라우저 안의 글쓰기 지원을 **개인용 편의 기능**에서 **기업 업무용 생산성 기능**으로 끌어올리는 변화입니다. 다만 현재는 출시가 일시 중단된 상태이므로, 기능 자체의 방향성은 주목하되 실제 배포와 사용자 안내는 Microsoft의 새 일정 공지를 확인한 뒤 진행하는 것이 좋겠습니다.
