# Agent 365 활동을 한눈에: Copilot Analytics에 Agent Dashboard가 추가됩니다

조직이 사용하는 에이전트가 늘어나면 단순한 인벤토리를 넘어 실제 활동과 활용 추세를 확인할 방법이 필요합니다. 어떤 에이전트가 사용되고 있는지, 시간이 지나며 활동이 어떻게 바뀌는지 알아야 투자·교육·거버넌스 결정을 내릴 수 있습니다.

Microsoft Viva의 Copilot Analytics(Insights)에 **Agent 365 Dashboard**가 추가됩니다.

> **한 줄 요약**
> Agent 365와 Copilot을 사용하는 조직은 Agent 365에 등록된 전체 에이전트의 활동과 과거 데이터를 Copilot Analytics에서 확인할 수 있습니다.

---

## 무엇이 새로워지나

Agent 365 데이터가 Copilot Analytics의 Agent Dashboard에 연결됩니다. 조직 리더는 Agent 365에 등록된 에이전트 전반의 활동을 이해하고, Dashboard에서 Historical Data를 확인할 수 있습니다.

이번 공지가 명시한 핵심 범위는 다음과 같습니다.

- Agent 365 등록 에이전트의 활동 가시성
- 조직 전체 에이전트 현황을 위한 Dashboard
- 과거 데이터 조회

## 왜 중요한가

에이전트 거버넌스는 등록과 차단만으로 끝나지 않습니다. 실제 사용 데이터를 함께 봐야 다음 질문에 답할 수 있습니다.

- 배포된 에이전트가 실제로 활용되는가
- 도입 활동이 시간에 따라 증가하는가
- 사용이 낮은 에이전트에 교육이나 개선이 필요한가
- 확산할 가치가 있는 에이전트는 무엇인가

Dashboard는 이런 판단을 위한 공통 데이터 화면을 제공합니다.

## 출시 일정

| 구분 | 일정 |
|---|---|
| Preview | 2026년 7월 |
| GA | 2026년 9월 |
| 대상 | Agent 365 및 Copilot 고객 |
| 위치 | Copilot Analytics의 Insights |

## 도입 관점 체크포인트

- Agent 365 Registry에 조직 에이전트가 빠짐없이 등록돼 있는지 확인합니다.
- Dashboard 접근 권한을 리더·AI 관리자·거버넌스 담당자 중 누구에게 부여할지 정합니다.
- 단순 활동량과 업무 가치·품질을 혼동하지 않도록 별도의 평가 지표를 함께 사용합니다.
- Historical Data의 보존 기간과 세부 측정 정의를 정식 문서에서 확인합니다.

## 마무리

Agent 365 Dashboard는 조직의 에이전트 운영을 감이 아니라 데이터로 관리하는 기반입니다. 인벤토리·보안 통제와 함께 활동 추세를 확인하면 어떤 에이전트를 개선·확산·정리할지 더 근거 있게 결정할 수 있습니다.
