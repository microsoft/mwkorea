# 기본 Planner 플랜에도 Agent Chat이 들어옵니다

Planner에서 원하는 작업을 찾거나 계획 상태를 확인하려면 필터와 버킷을 직접 살펴봐야 하는 경우가 많습니다. 작업이 많아질수록 "이번 주 지연된 업무가 무엇인지" 같은 간단한 질문에도 여러 화면을 확인하게 됩니다.

Microsoft Planner의 **Basic Plans에 Planner Agent Chat**이 추가됩니다.

> **한 줄 요약**
> 플랜 안에서 자연어로 질문하고, 관련 작업을 스마트하게 찾고, 작업을 관리할 수 있는 내장 Agent Chat이 제공됩니다.

---

## 핵심 기능

로드맵에서 안내한 기능은 세 가지입니다.

- **자연어 Q&A**: 플랜과 작업에 대해 일상 언어로 질문
- **Smart Task Discovery**: 조건과 맥락에 맞는 작업을 빠르게 탐색
- **In-plan Task Management**: Agent Chat을 벗어나지 않고 플랜의 작업 관리

## 활용 시나리오

- "이번 주 마감인데 완료되지 않은 작업을 보여줘."
- "내가 담당하는 우선순위 높은 업무를 찾아줘."
- "이 프로젝트에서 지연될 위험이 있는 작업은 무엇이야?"
- 대화 중 찾은 작업의 상태와 세부사항을 플랜 안에서 관리합니다.

실제 지원되는 질문과 변경 작업의 범위는 정식 제공 시 제품 문서에서 확인해야 합니다.

## 출시 일정

| 구분 | 내용 |
|---|---|
| GA | 2026년 8월 |
| 적용 대상 | Microsoft Planner Basic Plans |
| 기능 | 내장 Planner Agent Chat |

## 도입 관점 체크포인트

- Basic Plan과 Premium 기능의 차이를 사용자에게 명확히 안내합니다.
- 자연어 작업 변경이 가능한 범위와 실행 전 확인 절차를 점검합니다.
- 작업명·담당자·기한이 일관되게 관리돼야 Agent의 검색 품질도 좋아집니다.
- 팀별 Planner 사용 규칙과 Agent Chat 교육 자료를 함께 준비합니다.

## 마무리

Planner Agent Chat이 Basic Plans까지 확대되면 더 많은 사용자가 작업 목록을 직접 탐색하는 대신 대화로 계획을 이해하고 관리할 수 있습니다. 기존 Planner 데이터의 품질과 운영 규칙을 먼저 정리하면 Agent의 효과를 더 잘 활용할 수 있습니다.
