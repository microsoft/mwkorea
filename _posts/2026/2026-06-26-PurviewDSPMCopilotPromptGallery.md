# Purview DSPM에 Copilot Prompt Gallery가 온다: 보안 업무 프롬프트를 한곳에서

보안 Copilot 경험에서 중요한 것은 모델의 성능만이 아닙니다. 사용자가 **무엇을 어떻게 물어봐야 하는지** 아는 것도 그만큼 중요합니다. 특히 데이터 보안 업무는 역할과 시나리오가 다양합니다. 민감 정보 노출을 점검하는 사람, 정책을 만드는 사람, 리스크를 분석하는 사람, 사고 대응을 준비하는 사람마다 필요한 질문이 다릅니다.

Microsoft 365 Roadmap 항목 RM492618에 따르면, Microsoft Purview compliance portal의 **Data Security Posture Management(DSPM)** 영역에 **Copilot Prompt Gallery**가 제공됩니다. 이는 DSPM 안에서 Security Copilot 경험을 더 효과적으로 활용하도록 돕는, 선별된 프롬프트와 프롬프트북의 중앙 허브입니다.

사용자는 이 갤러리에서 프롬프트를 탐색하고 검색하며, 데이터 보안 워크플로를 더 빠르게 진행할 수 있습니다.

---

## 무엇이 새로운가

**DSPM Copilot Prompt Gallery**는 데이터 보안 업무에 맞춰 선별된 프롬프트와 프롬프트북 모음입니다. 사용자는 중앙 허브에서 관련 프롬프트를 빠르게 찾아 활용할 수 있고, 이를 통해 Security Copilot 경험을 더 자연스럽게 사용할 수 있습니다.

Microsoft가 설명한 핵심 가치는 다음과 같습니다.

- **중앙 집중형 허브**: DSPM에서 사용할 수 있는 프롬프트와 프롬프트북을 한곳에서 탐색
- **검색과 발견성 향상**: 역할과 사용 사례에 맞는 프롬프트를 더 쉽게 찾음
- **업무 흐름 간소화**: 반복적인 데이터 보안 작업을 프롬프트 기반으로 빠르게 수행
- **생산성 향상**: Security Copilot 경험의 활용도를 높여 작업 완료 시간을 단축

---

## DSPM과 Security Copilot을 잇는 실무형 경험

DSPM은 조직의 데이터 보안 상태를 이해하고 개선하는 데 초점을 둡니다. 하지만 좋은 도구가 있어도, 사용자가 매번 적절한 질문을 새로 만들어야 한다면 활용 장벽이 생깁니다.

Prompt Gallery는 이 장벽을 낮춥니다. 보안 담당자는 "어떤 질문을 해야 하는지"부터 고민하는 대신, 이미 준비된 프롬프트와 프롬프트북을 출발점으로 삼을 수 있습니다. 특히 데이터 보안 업무처럼 반복되는 패턴과 역할별 관점이 분명한 영역에서는, 잘 설계된 프롬프트 모음이 실제 생산성에 큰 영향을 줍니다.

---

## 활용 시나리오

- **민감 데이터 리스크 점검**: DSPM에서 드러난 데이터 보안 위험을 Copilot 프롬프트로 빠르게 요약·분석
- **정책 개선 검토**: 특정 보안 정책이나 레이블 적용 상태를 기준으로 개선 여지를 확인
- **역할별 업무 시작점 제공**: 데이터 보안 관리자, 컴플라이언스 담당자, 보안 운영 담당자가 각자의 업무에 맞는 프롬프트로 시작
- **반복 업무 표준화**: 자주 수행하는 조사·검토·보고 작업을 프롬프트북으로 재사용

## 도입 관점 체크포인트

- **보안팀 업무별 프롬프트 운영 체계**: 기본 갤러리만 사용하는 데서 멈추지 말고, 조직의 데이터 보안 프로세스에 맞는 추천 프롬프트 목록을 정리해 두면 좋습니다.
- **프롬프트북 표준화**: 반복되는 점검·분석·보고 작업은 프롬프트북 형태로 표준화해, 담당자별 편차를 줄일 수 있습니다.
- **Security Copilot 사용 권한 확인**: DSPM과 Security Copilot을 함께 사용하는 경험이므로, 관련 라이선스와 권한 모델을 사전에 확인해야 합니다.
- **결과 검토 절차 유지**: Copilot이 생성한 분석과 권고는 보안 의사결정의 출발점입니다. 최종 판단은 조직의 보안 담당자가 검토하도록 운영 절차를 마련하세요.

---

## 마무리

Copilot Prompt Gallery는 보안 담당자에게 "빈 프롬프트 입력창" 대신 **업무 맥락이 담긴 시작점**을 제공합니다. 데이터 보안 영역에서 Copilot의 가치를 높이려면, 모델을 도입하는 것만큼이나 좋은 질문과 반복 가능한 워크플로를 제공하는 것이 중요합니다. Purview DSPM의 Prompt Gallery는 그 방향을 보여 주는 실용적인 업데이트입니다.
