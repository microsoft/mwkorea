# Copilot Search 결과의 신뢰도를 높이는 방법: 권위 있는 콘텐츠 지정

조직 안에는 같은 주제를 다룬 문서가 여러 버전으로 흩어져 있는 경우가 많습니다. 오래된 초안, 개인이 정리한 메모, 공식 정책 문서가 뒤섞여 있으면 Copilot Search가 어떤 것을 우선 보여줘야 할지 모호해집니다.

Microsoft 365 admin center에 **권위 있는 콘텐츠(Authoritative content)** 관리 기능이 추가됩니다.

> **한 줄 요약**
> 특정 SharePoint 사이트를 권위 있는 콘텐츠로 지정하면 Microsoft 365 Copilot Search에서 해당 콘텐츠의 관련성과 순위가 높아집니다. admin center의 Copilot > Search > Authoritative content에서 최대 100개 사이트를 관리할 수 있습니다.

---

## 무엇이 새로워지나

관리자는 조직에서 공식적이고 신뢰할 수 있는 정보를 담은 SharePoint 사이트를 **권위 있는 콘텐츠**로 지정할 수 있습니다. 위치는 다음과 같습니다.

- Microsoft 365 admin center → Copilot → Search → Authoritative content
- 최대 100개의 SharePoint 사이트 추가·관리

이렇게 지정된 사이트의 콘텐츠는 Copilot Search 결과에서 관련성과 순위가 강화됩니다. 이 공지에는 `#copilotcontrolsystem` 태그가 붙어 있어 Copilot 제어 체계의 일부로 다뤄지고 있음을 알 수 있습니다.

## 왜 중요한가

검색 결과의 신뢰도는 결국 "어떤 출처를 우선하는가"에 달려 있습니다. 권위 있는 콘텐츠를 지정하면 다음을 기대할 수 있습니다.

- 공식 정책·표준 문서가 검색 상단에 노출
- 오래되거나 비공식적인 문서로 인한 혼란 감소
- 조직이 신뢰하는 정보 원본을 Copilot이 우선 참조

## 활용 시나리오

- 인사·법무·보안 정책을 담은 공식 SharePoint 사이트를 권위 있는 콘텐츠로 지정
- 제품 표준 문서 사이트를 우선순위로 올려 최신 규격이 먼저 노출되도록 설정
- 부서별 공식 지식 허브를 지정해 사내 검색 품질 향상

## 출시 일정

| 구분 | 내용 |
|---|---|
| GA | 2026년 9월 |
| 위치 | admin center → Copilot → Search → Authoritative content |
| 관리 한도 | 최대 100개 SharePoint 사이트 |

## 도입 관점 체크포인트

- 조직의 공식 정보가 어떤 SharePoint 사이트에 있는지 먼저 정리합니다.
- 권위 있는 콘텐츠 지정 기준과 관리 책임자를 정합니다.
- 100개 한도를 고려해 우선순위가 높은 사이트부터 선정합니다.
- 지정된 사이트의 콘텐츠 최신성을 주기적으로 점검합니다.

## 마무리

Copilot Search의 가치는 얼마나 많이 찾느냐가 아니라 얼마나 신뢰할 수 있는 결과를 먼저 보여주느냐에 있습니다. 권위 있는 콘텐츠 지정은 조직의 공식 정보를 검색 결과 전면에 세우는 실용적인 통제 수단입니다.
