# 관리자도 팀의 에이전트 활동을 확인합니다: Agent 365 Dashboard for Managers

앞서 Copilot Analytics(Insights)에 Agent 365 Dashboard가 추가되어 리더가 조직 전체의 에이전트 활동을 확인할 수 있게 되었습니다. 이번 업데이트는 그 대상을 **관리자(Managers)**까지 확장합니다.

> **한 줄 요약**
> 관리자도 Copilot Analytics의 Agent Dashboard에서 Agent 365에 등록된 에이전트 활동과 과거 데이터를 확인할 수 있습니다.

---

## 무엇이 새로워지나

이번 업데이트로 조직의 관리자가 Copilot Analytics(Insights)의 **Agent Dashboard**를 통해 Agent 365 데이터에 접근할 수 있습니다. Agent 365와 Copilot을 사용하는 고객은 Agent 365 Dashboard를 볼 수 있고, Agent Dashboard에서 **과거 데이터(historical data)**도 확인할 수 있습니다.

즉, 조직 전체를 보는 리더 관점의 대시보드에 더해, 팀을 이끄는 관리자도 자신의 범위에서 에이전트 활동을 파악할 수 있게 됩니다.

## 왜 중요한가

에이전트 도입 관리가 리더 계층에만 머무르면 실제 팀 단위의 활용 개선으로 이어지기 어렵습니다. 관리자가 직접 활동 데이터를 볼 수 있으면 다음이 가능해집니다.

- 팀 내 에이전트 활용도 파악
- 활용이 저조한 영역에 대한 교육·안내 결정
- 도입 성과를 팀 단위로 추적

## 활용 시나리오

- 관리자가 팀의 에이전트 활동 추세를 정기적으로 검토
- 특정 에이전트의 활용이 낮은 팀에 지원 제공
- 팀 단위 도입 목표와 실제 활동 비교

## 출시 일정

| 구분 | 내용 |
|---|---|
| GA | 2026년 8월 |
| 대상 | Agent 365 및 Copilot 고객 (관리자) |
| 위치 | Copilot Analytics(Insights)의 Agent Dashboard |

## 도입 관점 체크포인트

- 관리자에게 제공되는 데이터 범위와 접근 권한을 명확히 정의합니다.
- 관리자가 보는 팀 단위 데이터와 개인정보 보호 기준의 균형을 점검합니다.
- 단순 활동량과 업무 가치·품질을 구분하는 지표를 함께 사용합니다.
- 리더용 대시보드와 관리자용 대시보드의 역할 차이를 안내합니다.

## 마무리

Agent 365 Dashboard가 관리자까지 확대되면서, 조직은 리더의 전사 관점과 관리자의 팀 관점을 함께 활용해 에이전트 도입을 더 세밀하게 관리할 수 있게 됩니다. 데이터를 근거로 팀 단위 개선을 이끄는 기반이 마련되는 셈입니다.
