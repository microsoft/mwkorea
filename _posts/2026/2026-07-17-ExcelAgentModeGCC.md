# Excel Agent Mode, GCC 환경에도 제공됩니다

Excel에서 예산을 정리하거나 재무 모델을 만들 때, 어떤 함수를 써야 할지, 표와 피벗테이블을 어떻게 구성할지 고민하는 데 시간이 많이 듭니다. Agent Mode는 이 작업을 Copilot과 나란히 진행하도록 돕습니다.

상용 환경에서 제공되던 **Excel의 Agent Mode(Edit with Copilot)**가 이번 업데이트로 **GCC(Government Community Cloud) 환경**에도 제공됩니다.

> **한 줄 요약**
> Edit with Copilot을 통해 Copilot과 나란히 통합 문서를 만들고 편집할 수 있으며, Agent Mode는 표·차트·피벗테이블·수식 같은 Excel의 강력한 도구를 활용해 작업을 완료하도록 돕습니다. 이 기능이 GCC 환경으로 확대됩니다.

---

## 무엇이 새로워지나

이번 업데이트의 핵심은 **범위 확대**입니다. 상용 테넌트에서 제공되던 Excel Agent Mode가 규제 요건이 다른 GCC 환경까지 적용됩니다.

Agent Mode는 다음과 같은 작업을 지원합니다.

- 예산 업데이트
- 재무 모델 작성
- 데이터 분석

이 과정에서 Excel의 표(tables), 차트(charts), 피벗테이블(PivotTables), 수식(formulas) 같은 핵심 도구를 활용합니다.

## 왜 중요한가

GCC 환경은 상용 테넌트와 분리된 보안·규정 준수 요건을 따르므로, 상용에서 제공되는 기능이 같은 시점에 제공된다고 가정할 수 없습니다. 이번 발표는 정부·공공 고객도 Excel에서 Copilot과 나란히 작업하는 Agent Mode 경험을 사용할 수 있게 된다는 점에서 의미가 있습니다.

## 활용 시나리오

- 부서 예산안을 Copilot과 함께 단계별로 수정
- 재무 모델의 시나리오를 표와 수식으로 구성
- 대량 데이터를 피벗테이블로 요약·분석
- 차트로 결과를 시각화하며 검토

## 출시 일정

| 구분 | 내용 |
|---|---|
| GA | 2026년 10월 |
| 대상 환경 | GCC |
| 기능 | Excel Agent Mode (Edit with Copilot) |

## 도입 관점 체크포인트

- GCC 환경의 라이선스와 Copilot 사용 조건을 확인합니다.
- Agent Mode가 생성·수정한 결과는 사람이 검토하는 절차를 둡니다.
- 민감한 재무·예산 데이터 처리에 대한 조직 정책을 점검합니다.
- 사용자 교육 자료에 Agent Mode 활용법을 반영합니다.

## 마무리

Excel Agent Mode의 GCC 확대로 정부·공공 고객도 상용 고객과 동등한 수준의 Copilot 기반 스프레드시트 작업 경험을 사용할 수 있게 됩니다. 2026년 10월 GA를 앞두고 환경별 제공 조건을 미리 확인해 두시길 권합니다.
