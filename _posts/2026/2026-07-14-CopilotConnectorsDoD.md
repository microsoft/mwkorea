# 미 국방부 환경에서도 Copilot 커넥터를 사용할 수 있습니다

조직의 업무 지식은 Microsoft 365 안에만 있지 않습니다. 각종 업무 시스템과 외부 데이터 소스에 흩어진 콘텐츠를 Copilot이 활용하려면, 해당 데이터를 Microsoft Graph로 안전하게 연결할 수 있어야 합니다.

이번 Microsoft 365 로드맵 업데이트에 따르면 **Copilot 커넥터가 미국 Department of Defense(DoD) 환경에도 제공**됩니다.

> **한 줄 요약**
> DoD 환경의 조직도 외부 데이터 소스 콘텐츠를 Microsoft Graph에 통합해 Copilot의 지능, 맥락 기반 응답, 콘텐츠 검색·발견 범위를 확장할 수 있습니다.

---

## Copilot 커넥터가 하는 일

Copilot 커넥터는 조직이 사용하는 외부 데이터 소스의 콘텐츠를 Microsoft Graph에 연결합니다. 이렇게 연결된 콘텐츠는 권한을 기반으로 검색과 Copilot 응답의 근거로 활용될 수 있습니다.

공지에서 강조한 기대 효과는 다음과 같습니다.

- **Copilot의 지능 확장**: Microsoft 365 밖에 있는 업무 지식을 답변에 활용
- **맥락 기반 응답 개선**: 조직 고유 데이터에 근거한 더 관련성 높은 답변 제공
- **콘텐츠 발견성 향상**: 여러 시스템에 흩어진 정보를 Microsoft 환경에서 더 쉽게 찾기

## DoD 환경 확대의 의미

DoD 환경은 일반 상용 테넌트와 분리된 보안·규정 준수 요구사항을 따릅니다. 따라서 상용 환경에서 제공되는 기능이 같은 시점과 범위로 제공된다고 가정할 수 없습니다.

이번 발표는 Copilot을 위한 외부 데이터 연결 역량이 규제가 엄격한 정부 클라우드 환경으로 확대된다는 점에서 의미가 있습니다.

## 제공 일정

| 항목 | 내용 |
|---|---|
| 대상 환경 | U.S. Department of Defense |
| 기능 | Copilot 커넥터 |
| GA | 2026년 6월로 안내 |

실제 커넥터별 지원 범위와 테넌트 노출 시점은 admin center 및 관련 문서에서 별도로 확인해야 합니다.

## 도입 관점 체크포인트

- DoD 환경에서 지원되는 커넥터 목록과 기능 범위를 확인합니다.
- 외부 데이터 원본의 인증·권한·보존 정책이 조직의 규정과 일치하는지 점검합니다.
- 연결 전 소규모 데이터 세트로 검색 정확도와 권한 반영을 검증합니다.
- 커넥터 운영 담당자와 보안·컴플라이언스 담당자의 책임 범위를 정리합니다.

## 마무리

Copilot 커넥터의 DoD 지원은 정부·국방 조직이 기존 보안 경계를 유지하면서도 외부 업무 지식을 Copilot에 연결할 수 있는 기반을 넓혀 줍니다. 실제 도입 전에는 환경별 지원 범위와 권한 모델을 우선 확인하시길 권합니다.
