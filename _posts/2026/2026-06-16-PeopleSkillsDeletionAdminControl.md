# People Skills 데이터를 완전히 지운다 — 관리자 전용 영구 삭제 컨트롤 출시 예정

Microsoft 365의 People Skills는 Copilot과 Viva 같은 M365 경험에서 직원 스킬을 자동으로 추론·노출하는 기능입니다. 프로필 카드에 표시되는 스킬 데이터는 편리하지만, 조직이 이 기능을 중단하거나 오프보딩해야 할 때 데이터를 어떻게 처리할지 명확하지 않았습니다.

이번 업데이트로 관리자는 테넌트 전체의 People Skills 데이터를 완전히, 영구적으로 삭제할 수 있는 전용 컨트롤을 갖게 됩니다. 데이터 거버넌스 요구사항이 엄격한 조직, 또는 People Skills 사용을 중단하려는 조직에 실질적인 통제권을 제공하는 중요한 변화입니다.

---

## 무엇이 달라지나

기존에는 People Skills를 비활성화해도 이미 수집된 스킬 데이터가 테넌트에 남아 있었습니다. 이제 관리자는 People Skills 설정 페이지에서 **영구 삭제(Permanent Removal)** 컨트롤을 통해 다음 데이터를 한 번에 지울 수 있습니다.

- **스킬 라이브러리(Skills Library) 전체** — 조직이 관리하는 스킬 목록
- **사용자 확인(Confirmed) 스킬** — 직원이 직접 확인한 스킬
- **추론(Inferred) 스킬** — AI가 업무 패턴에서 자동 추론한 스킬
- **가져온(Imported) 스킬** — 외부 시스템에서 동기화한 스킬

삭제가 시작되면 스킬은 프로필 카드에 더 이상 표시되지 않으며, Copilot과 Viva 등 스킬 기반 M365 경험에도 활용되지 않습니다.

## 핵심 포인트

> 이 컨트롤은 **관리자 전용**입니다. 자동 변경은 없으며, 관리자가 명시적으로 실행해야 삭제가 시작됩니다.

| 항목 | 내용 |
|---|---|
| 대상 데이터 | 스킬 라이브러리 + 사용자 전체 스킬(확인·추론·가져온) |
| 실행 주체 | 관리자(Admin)만 가능 |
| 자동 실행 여부 | 없음 — 관리자가 직접 트리거 |
| 삭제 후 효과 | 프로필 카드·Copilot·Viva 스킬 경험에서 스킬 미노출 |

## 활용 시나리오

- **규정 준수(Compliance) 요구**: GDPR 등 데이터 보호 규정에 따라 직원 스킬 데이터를 보유할 수 없는 경우
- **People Skills 오프보딩**: 조직이 People Skills 기능 사용을 중단하고 데이터를 완전히 초기화하려는 경우
- **데이터 거버넌스 감사 대응**: 테넌트 내 AI 추론 데이터의 존재를 감사 항목으로 관리해야 하는 경우

## 출시 일정

| 항목 | 내용 |
|---|---|
| 정식 출시(GA) | 2026년 7월 |
| 적용 범위 | Microsoft 365 전체 테넌트 (관리자 설정 페이지) |

## 도입 관점 체크포인트

- 이 컨트롤은 **되돌릴 수 없는 영구 삭제**입니다. 실행 전 반드시 데이터 백업·내보내기 여부를 검토하십시오.
- 아직 People Skills를 활성화하지 않은 조직은 이번 변화와 무관합니다.
- 컴플라이언스 팀과 IT 관리팀이 협력해 삭제 실행 정책과 승인 프로세스를 사전에 정의해 두는 것이 좋습니다.

## 마무리

People Skills 영구 삭제 컨트롤은 AI 추론 데이터에 대한 조직의 **자기결정권**을 강화하는 중요한 거버넌스 기능입니다. 2026년 7월 출시를 앞두고 현재 스킬 데이터 현황과 컴플라이언스 요구사항을 미리 점검해 두시기 바랍니다.
