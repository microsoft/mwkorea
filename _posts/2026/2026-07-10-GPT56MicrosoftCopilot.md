# 오늘부터 이용 가능: OpenAI GPT-5.6이 Microsoft 365 Copilot에 들어옵니다

오늘부터 **OpenAI의 GPT-5.6 모델군**이 Word, Excel, PowerPoint, Chat, 그리고 Cowork를 아우르는 Microsoft 365 Copilot 전반에서 이용 가능해집니다. OpenAI와 Microsoft는 함께 협력해 GPT-5.6을 Microsoft 365 앱 전반의 지식 노동에 맞게 최적화했고, 그 결과 이 모델은 **Microsoft 365 Copilot의 선호 모델(preferred model)**로 자리 잡게 되었습니다. 모델을 네이티브로 제공하는 것에 더해, Microsoft는 API를 통해 OpenAI 모델에 직접 접근하는 방식으로도 GPT-5.6을 Microsoft 365 고객에게 제공합니다.

지식 노동은 복잡합니다 — 여러 단계에 걸쳐 있고, 모호하며, 여러 도구와 파일에 흩어져 있습니다. 에이전틱(agentic)하고 다단계인 작업에 대한 더 강력한 추론 능력을 갖춘 GPT-5.6은 Copilot이 이런 복잡성을 더 효율적으로 처리하도록 도와, 초기 아이디어에서 더 완성된 결과물로 더 빠르게 나아가게 합니다. Microsoft 365 Copilot은 업무를 위해 만들어진 AI이며, GPT-5.6 같은 프론티어 모델은 Work IQ, Microsoft 365 앱, 그리고 고객이 Microsoft에 기대하는 엔터프라이즈급 보안·컴플라이언스·개인정보 보호를 아우르는 더 큰 시스템의 중요한 한 부분입니다.

---

## 매일 쓰는 앱 전반에서 더 높은 품질의 작업

GPT-5.6은 Word, Excel, PowerPoint에서 일상 업무의 품질을 끌어올려, 거친 의도에서 더 탄탄한 초안, 더 명확한 분석, 더 풍부한 프레젠테이션 초안으로 나아가도록 돕습니다.

**Word에서** Copilot은 거친 아이디어를 더 완성도 있는 초안으로 만들고, 더 탄탄한 구조와 흐름으로 콘텐츠를 정리하며, 글을 더 다듬어진 결과물로 발전시키는 데 도움을 줍니다.

**Excel에서** GPT-5.6은 Copilot이 더 복잡한 분석을 추론하며, 수작업 조립을 줄이고 작업(task)에서 결과(outcome)로 더 빠르게 나아가도록 돕습니다.

**PowerPoint에서** Copilot은 더 탄탄한 슬라이드 콘텐츠, 더 나은 시각적 균형, 다양한 프레젠테이션 스타일에 걸친 더 큰 유연성을 갖춘 더 풍부한 프레젠테이션 초안을 만들 수 있습니다.

이 앱들 전반에 걸쳐, GPT-5.6은 지식 노동자가 최종 결과물에 대한 통제권을 유지하면서도 Copilot이 아이디어에서 완성된 작업 사이의 더 많은 단계를 대신 처리하도록 돕습니다.

### Chat에서 복잡한 질문을 추론하다

Copilot Chat에서 GPT-5.6은 옵션 비교, 계획 구조화, 문제 해결, 모호함 정리처럼 더 복잡한 질문을 추론하도록 Copilot을 돕습니다. 이를 통해 열린 형태의 프롬프트가 더 명확하고 실행 가능한 응답으로 바뀝니다.

### Copilot Cowork에서 복잡한 작업을 완료하다

GPT-5.6은 Copilot Cowork의 에이전틱 작업도 강화하여, 복잡한 다단계 작업을 지시에서 완료된 결과까지 이어 줍니다. 사용자가 원하는 결과를 설명하면, Cowork는 여러 도구와 파일에 걸쳐 계획을 세우고 추론하며 작업을 처음부터 끝까지 실행해, 단순한 초안이나 권고안이 아니라 **완성된 결과물**을 돌려줍니다. GPT-5.6을 통해 Cowork는 더 효율적으로 실행할 수 있고, 처음부터 더 완성도 있고 다듬어졌으며 바로 쓸 수 있는 결과물을 만들어 냅니다. 비즈니스가 실제로 운영되는 시스템에 기반한 **Work IQ**와 결합하여, Cowork는 각 단계에 알맞은 맥락을 가져와 작업이 여러분의 파일·도구·비즈니스 맥락을 반영하도록 합니다. 결과를 조립하는 데 쓰는 시간은 줄고, 그 결과로 무엇을 할지 결정하는 데 쓰는 시간은 늘어납니다.

## 오늘 바로 시작하기

업무가 이루어지는 방식의 변화는 실재하며, 지금 하려는 작업에 맞는 모델을 고르는 일이 중요합니다.

GPT-5.6은 Word, Excel, PowerPoint, Chat, Copilot Cowork의 Copilot에 순차적으로 제공됩니다. 제공이 시작되면 Copilot은 작업에 가장 적합할 때 GPT-5.6을 자동으로 사용할 수 있습니다. 모델 선택이 가능한 곳에서는 사용자가 모델 선택기(model selector)에서 GPT-5.6을 직접 선택할 수도 있습니다.

## 제공 범위 및 참고 자료

| 구분 | 내용 |
|------|------|
| **제공 앱** | Word, Excel, PowerPoint, Copilot Chat, Copilot Cowork |
| **제공 방식** | 네이티브 통합 + OpenAI API를 통한 직접 접근 |
| **모델 선택** | 자동 사용 또는 모델 선택기에서 수동 선택(제공되는 경우) |

제공 범위는 지역과 테넌트 구성에 따라 다를 수 있습니다. 최신 롤아웃 현황은 [Microsoft Learn 문서](https://go.microsoft.com/fwlink/p/?LinkId=2369720), [Microsoft 365 Roadmap](https://www.microsoft.com/en-us/microsoft-365/roadmap), [Microsoft 365 Copilot 릴리스 노트](https://learn.microsoft.com/en-us/microsoft-365/copilot/release-notes?tabs=all)를 참고하시기 바랍니다. 모델 자체에 대한 자세한 내용은 [OpenAI의 GPT-5.6 발표](https://openai.com/index/gpt-5-6-preferred-model-microsoft-365-copilot/)에서 확인할 수 있습니다.

## 도입 관점 체크포인트

- 조직의 테넌트 구성에 따라 GPT-5.6 제공 시점이 다를 수 있으므로, admin center와 릴리스 노트를 통해 실제 적용 여부를 확인
- 모델 선택기가 제공되는 앱에서는 업무 성격에 따라 GPT-5.6과 기존 모델 중 어떤 것이 더 적합한지 팀 단위로 비교
- Cowork처럼 에이전틱 작업이 늘어나는 만큼, 결과물 검토·승인 프로세스를 함께 정비할 필요가 있는지 검토

## 마무리

GPT-5.6이 Microsoft 365 Copilot의 선호 모델로 자리 잡으면서, Word·Excel·PowerPoint 같은 일상 도구부터 Cowork의 복잡한 에이전틱 작업까지 전반적인 품질과 효율이 함께 올라갈 것으로 기대됩니다. 오늘부터 순차적으로 제공되는 만큼, 조직의 롤아웃 현황을 확인하고 새 모델을 직접 경험해 보시길 권합니다.
