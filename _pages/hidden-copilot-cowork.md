<h2 style="margin-top: 1.4rem; padding: 0.55rem 0.85rem; border-left: 6px solid #0f766e; background: #f8fafc;">1️⃣ 새로운 AI 도구 – Agentic AI, 업무를 '실행'하는 지능</h2>

<p>우리는 지금까지 AI를 조언자처럼 썼습니다. "요약해줘", "초안 만들어줘", "정리해줘"—AI는 답을 주고, 우리는 그걸 받아서 직접 실행했죠.</p>

<p>하지만 <strong>Agentic AI</strong>는 다릅니다. 목표만 말하면, AI가 계획을 세우고 → 여러 단계를 실행하고 → 결과를 확인받는 방식으로 일을 끝까지 처리합니다. 마치 신뢰할 수 있는 팀원에게 업무를 위임하듯, AI는 이제 실행의 주체가 됩니다.</p>

<p>Microsoft 365 <strong>Copilot Cowork</strong>는 바로 이 Agentic AI 경험을 엔터프라이즈 업무 환경에 적용한 도구입니다.</p>

<div style="margin: 1rem 0 1.4rem; padding: 1rem 1.1rem; border-radius: 16px; background: #fefce8; border: 1px solid #fde68a; color: #854d0e;">
  <div style="font-weight: 800; margin-bottom: 0.35rem;">🧪 Research Preview</div>
  <div>Copilot Cowork는 현재 Preview(Frontier Program) 단계입니다. 기능은 발전 중이며, 단계적으로 확장되고 있습니다.</div>
  <div style="margin-top: 0.5rem;">👉 바로가기: <a href="https://aka.ms/cowork" target="_blank">https://aka.ms/cowork</a></div>
</div>

<h2 style="margin-top: 1.4rem; padding: 0.55rem 0.85rem; border-left: 6px solid #1d4ed8; background: #f8fafc;">3️⃣ 어떻게 활용할 수 있을까요?</h2>

<p>Copilot Cowork는 일상적이고 반복적인 업무를 위임함으로써, 여러분이 더 전략적이고 창의적인 일에 집중할 수 있도록 돕습니다.</p>

<div style="display: grid; grid-template-columns: 1fr; gap: 14px; margin: 1rem 0 1.4rem;">
  <div style="padding: 1rem 1.1rem; border-radius: 14px; background: #ecfeff; border: 1px solid #99f6e4; color: #115e59;">
    <div style="font-weight: 800; margin-bottom: 0.35rem;">📧 이메일 관리</div>
    <div>"오늘 받은 이메일들을 중요도별로 구분해서 Word 파일로 정리해줘"</div>
    <div style="margin-top: 0.4rem; font-size: 0.92rem; color: #6b7280;">→ 받은편지함 검색 → 중요도 분류 → 요약 제공</div>
  </div>
  <div style="padding: 1rem 1.1rem; border-radius: 14px; background: #eff6ff; border: 1px solid #bfdbfe; color: #1d4ed8;">
    <div style="font-weight: 800; margin-bottom: 0.35rem;">📅 일정 관리</div>
    <div>"[사람들]이랑 다음 주 중 30분 미팅 잡아줘. 주제는 [내용]"</div>
    <div style="margin-top: 0.4rem; font-size: 0.92rem; color: #6b7280;">→ 참석자 일정 확인 → 가능한 시간대 제안 → 사용자 승인 → Outlook 초대장 발송</div>
  </div>
  <div style="padding: 1rem 1.1rem; border-radius: 14px; background: #f5f3ff; border: 1px solid #c4b5fd; color: #5b21b6;">
    <div style="font-weight: 800; margin-bottom: 0.35rem;">📊 문서 제작</div>
    <div>"[주제]로 보고용 PT 만들어줘. 현황·문제점·제안 순서로"</div>
    <div style="margin-top: 0.4rem; font-size: 0.92rem; color: #6b7280;">→ 관련 자료 수집 → 슬라이드 구조 설계 → 초안 작성 → 사용자 승인 → PowerPoint 저장 및 공유</div>
  </div>

    <div style="margin-top: 0.4rem; font-size: 0.92rem; color: #6b7280;">→ Outlook 일정 확인 → 중요 이메일 요약 → 요약본 생성 → 사용자 승인 → Outlook 전달</div>

<h2 style="margin-top: 1.4rem; padding: 0.55rem 0.85rem; border-left: 6px solid #7c3aed; background: #f8fafc;">4️⃣ 예시 데모 시나리오</h2>

<p>실제 업무 흐름 그대로, 엔드투엔드로 실행됩니다.</p>

<div style="margin: 1rem 0 1.4rem; padding: 1rem 1.1rem; border-radius: 16px; background: #f5f3ff; border: 1px solid #c4b5fd; color: #5b21b6;">
  <div style="font-weight: 800; margin-bottom: 0.45rem;">🎬 시나리오 1: 미팅 준비부터 일정 등록, 요약 자료 생성, 메일 발송까지</div>
  <div style="line-height: 1.85; color: #374151;">"[파일]과 관련해서 [사람]과 미팅을 하려고 해. 미팅 안건을 정리해주고, [사람]과 내가 모두 가능한 시간에 일정을 가장 빠른 일자로 잡아줘. 일정에는 첨부 파일에 대한 간단한 내용도 함께 포함해줘."</div>
  <ul style="margin: 0.7rem 0 0; padding-left: 1.2rem; line-height: 1.85; color: #374151;">
    <li>첨부된 Word 파일 내용을 기반으로 미팅 목적 및 안건 정리</li>
    <li>참석자 전원의 일정 고려하여, 가장 빠른 공통 가능 시간으로 일정 자동 생성</li>
    <li>첨부 파일을 요약하여 안건을 포함한 메일 초안 작성</li>
  </ul>
</div>

<div style="margin: 1rem 0 1.4rem; padding: 1rem 1.1rem; border-radius: 16px; background: #ecfeff; border: 1px solid #99f6e4; color: #115e59;">
  <div style="font-weight: 800; margin-bottom: 0.45rem;">🎬 시나리오 2: Viva Engage 게시글 확인 후 PDF 문서 만들기</div>
  <div style="line-height: 1.85; color: #374151;">"Viva Engage 커뮤니티에 'Start your day with Agent'로 포스팅된 게시글들을 파일로 만들어서 공유하고 싶어. Day 1~9까지 시리즈로 포스팅되어 있는 게시글을 순서대로 모아서 하나의 PDF 파일로 생성해. 완성된 PDF를 첨부하여 고객에게 이메일로 전송할 수 있도록 이메일 초안을 작성해서 나에게 메일로 보내줘."</div>
</div>

<h2 style="margin-top: 1.4rem; padding: 0.55rem 0.85rem; border-left: 6px solid #ea580c; background: #f8fafc;">5️⃣ 기대 효과</h2>

<div style="display: grid; grid-template-columns: 1fr; gap: 14px; margin: 1rem 0 1.4rem;">
  <div style="padding: 1rem 1.1rem; border-radius: 14px; background: #ecfeff; border: 1px solid #99f6e4; color: #115e59;">
    <div style="font-weight: 800; margin-bottom: 0.35rem;">⏱️ 시간 절약과 맥락 전환 감소</div>
    <div>여러 앱을 오가며 반복 작업을 직접 처리하던 시간이 사라집니다. Cowork에 일을 맡기고, 여러분은 더 중요한 의사결정에 집중하세요.</div>
  </div>
  <div style="padding: 1rem 1.1rem; border-radius: 14px; background: #eff6ff; border: 1px solid #bfdbfe; color: #1d4ed8;">
    <div style="font-weight: 800; margin-bottom: 0.35rem;">📈 결과물 품질 향상</div>
    <div>Cowork는 Work IQ를 통해 조직 전체의 맥락을 이해하므로, 정확하고 일관성 있는 결과물을 만들어냅니다. 참고할 Reference 문서나 PPT를 첨부하면 더 완성도 높은 초안을 받을 수 있습니다.</div>
  </div>
  <div style="padding: 1rem 1.1rem; border-radius: 14px; background: #f5f3ff; border: 1px solid #c4b5fd; color: #5b21b6;">
    <div style="font-weight: 800; margin-bottom: 0.35rem;">🎯 사용자 통제 유지</div>
    <div>AI가 독립적으로 실행하지만, 모든 중요 단계마다 사용자 승인을 요청하므로 안심하고 업무를 위임할 수 있습니다.</div>
  </div>
  <div style="padding: 1rem 1.1rem; border-radius: 14px; background: #fff7ed; border: 1px solid #fdba74; color: #c2410c;">
    <div style="font-weight: 800; margin-bottom: 0.35rem;">🔄 업무 위임의 새로운 패러다임</div>
    <div>Cowork는 "빠른 처리"가 목적이 아니라 업무 위임이 핵심입니다. 사용자의 요청을 여러 단계의 작업으로 나눠 장시간에 걸쳐 수행하기 때문에, 결과를 기다리기보다는 업무를 위임하고 다른 업무에 집중하는 시간으로 활용할 수 있습니다.</div>
  </div>
</div>
